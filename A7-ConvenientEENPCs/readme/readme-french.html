Présentation

Un mod pour BG:EE, SoD, BG2:EE et EET.

Les nouveaux personnages recrutables apparus dans BG:EE et BG2:EE ne sont pas au goût de tout le monde. Ce mod propose plusieurs options pour les adapter à votre convenance. Vous pouvez rendre leur première apparition dans le jeu moins intrusive, en vous permettent de contrôler si vous acceptez d'établir un premier contact, et quand. Pour ce faire, quelques événements mineurs ont été légèrement modifiés.

Sinon, vous pouvez aussi désactiver définitivement tous les PNJ du jeu, ou seulement certains. Dans ce cas, vous pouvez aussi décider de rendre utilisables par quiconque leurs objets personnels ou ceux obtenus dans leur quête. Vous les trouverez soit dans des boutiques, soit sur des cadavres.

Des composants supplémentaires vous permettent d'améliorer leurs portraits et de personnaliser leurs classes et/ou leurs profils, selon vos envies. Vous trouverez des informations complémentaires plus bas.

N'hésitez pas à consulter les forums de SHS ou de Beamdog pour poser des questions, et à télécharger la dernière version du mod soit sur le site Spellhold Studios, soit sur GitHub.

HAUT DE PAGE

Compatibilité

« Convenient Enhanced Edition NPCs » devrait être compatible avec tous les mods qui ne modifient pas de manière significative les positions d'apparition et les scripts des PNJ des Éditions Améliorées. À l'heure actuelle, seul le mod de Pecca « EE_content_tweaks » entre en conflit avec lui dans BG:EE. C'est pourquoi, si ce dernier a été installé, ce mod refusera d'installer les composants incompatibles.

Il est entièrement compatible avec Siege of Dragonspear. Cependant, les options pour modifiers les PNJ de SoD sont plus restreintes en raison de leur étroite implication dans l'intrigue.

IMPORTANT : La version 4.0 de ce mod contient plusieurs changements structurels. Ne l'installez pas conjointement avec des versions précédentes (3.5 ou précédentes) !

HAUT DE PAGE

Installation

Il s'agit d'un mod WeiDU, ce qui signifie qu'il est très facile à installer. Il vous suffit d'extraire les fichiers de l'archive dans votre répertoire de jeu (le dossier qui contient le fichier « chitin.key »), d'exécuter « setup-A7-ConvenientEENPCs.exe » (Windows) ou « setup-A7-ConvenientEENPCs.command » (macOS) et de suivre les instructions affichées à l'écran.

Je vous recommande fortement de commencer une nouvelle partie une fois ce mod installé.

Pour le désinstaller, lancez « setup-A7-ConvenientEENPCs.exe » ou « setup-A7-ConvenientEENPCs.command » une nouvelle fois et suivez les instructions.

Note spéciale pour Siege of Dragonspear fourni par Steam/GOG

Good Old Games (GOG) et Steam fournissent le contenu de Siege of Dragonspear dans un format que WeiDU, l'outil utilisé pour installer ce mod, ne peut pas reconnaitre. Vous devez donc lancer le programme DLC Merger dans votre répertoire de SoD avant d'installer ce mod, ou tout autre mod utilisant WeiDU.
