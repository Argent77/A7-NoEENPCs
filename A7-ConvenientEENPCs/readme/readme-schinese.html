概述

本模组适用于BG:EE、BG2:EE和EET。

增强版新增队友与原版风格有些不同，可能并不符合每个人的口味。本模组提供了一些选项，可以让你按自己的舒适度调整。你可以让队友首次出现时表现得更加被动，从而对是否或何时与其接触有更多的控制权。少数事件将略有改变以实现此目的。

另外也可以简单的从游戏中移除一个或多个队友，这种情况下你可以选择让他们的个人物品或任务物品在商店里或作为战利品出现，从而让所有人都能使用。

另一个组件可以让你改善队友的肖像，或者按自己喜好改变他们的职业和宗派，具体详见下文。

如有问题请访问SHS论坛或Beamdog论坛，也可从Spellhold Studios或GitHub下载最新版本。

兼容性

“增强版队友便利性调整”应与现有的所有模组兼容，只要是不改变增强版队友起始位置和脚本的，均可以兼容。目前，只有Pecca的模组“EE_content_tweaks”已知会在BG:EE中干扰本模组。本模组将检查其是否已安装，如果检测到则拒绝安装相关组件。

本模组与《龙矛围攻》完全兼容，但是由于SoD中队友与剧情关联比较紧密，其调整选项将受到更多限制。

重要提示：本模组4.0版包含多项结构性变化，请勿将其与本模组的早期版本（3.5或更早版本）一起安装！

安装

这是个WeiDU模组，安装过程非常简单，只需将下载的压缩包解压到游戏安装目录（"chitin.key"文件所在目录），运行"setup-A7-ConvenientEENPCs.exe" (Windows)或"setup-A7-ConvenientEENPCs.command" (macOS)，然后按照指令操作即可开始安装。

强烈建议安装本模组后重新开始游戏。

若要卸载，再次运行"setup-A7-ConvenientEENPCs.exe"或"setup-A7-ConvenientEENPCs.command"并按提示操作。

关于《龙矛围攻》（SoD）的说明

GOG和Steam都以无法开箱即用的模式安装《龙矛围攻》扩展包。你必须先安装一个名为DLC Merger的模组，然后才能安装本模组或任何其他基于WeiDU的模组。

  1. 修改增强版队友(BG:EE、BG:SOD、BG2:EE和EET)

    1. 禁用所有队友：一次性禁用所有队友。
    2. 将所有队友设为被动：一次性将所有队友设为被动。
    3. 为每个队友单独决定：见下文。

  2. 单独修改：贝洛斯

    1. 禁用贝洛斯（仅限BG:EE和EET）：本组件禁止贝洛斯在BG:EE中出现。（《死斗黑坑》和《龙矛围攻》不受影响。）
    2. 将贝洛斯设为被动（仅限BG:EE和EET）：本组件将贝洛斯出场动画减少到最低限度，并禁止其自行发起对话。

    本组件禁止多恩在BG:EE、BG:SOD和BG2:EE中出现。

    1. Disable Hexxat (BG2:EE and EET only): This component prevents Hexxat and related NPCs from appearing in BG2:EE.
    2. Make Hexxat passive (BG2:EE and EET only): This component prevents Hexxat from initiating a conversation on her own.

  5. Modify individually: Neera

    1. Disable Neera (BG:EE, BG:SOD, BG2:EE and EET): This component prevents Neera from appearing in BG:EE, BG:SOD and BG2:EE.

In BG2:EE this component triggers the introduction cutscene in the Bridge District only when you talk to Neera. She can be found near the place where the Red Wizards cutscene occurs after the townsfolk talked about the local murders.

  8. Allied Caelar Argent in Avernus boss battle (BG:SOD and EET)

    This component allows you to control Caelar Argent if she joins the battle as an ally. The following options allow you to refine her behavior even further:

    1. Joins as controllable follower if no free party slots available: Retains the original behavior of the game by letting Caelar Argent join as a regular party member if a free slot is available.
    2. Always joins as controllable follower: Prevents Caelar Argent from joining the party when invited, even if there is a free party slot available. This option is mostly useful for players who want to solo the whole game.
